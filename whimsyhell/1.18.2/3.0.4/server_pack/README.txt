Prerequisite: Java 17 Runtime Environment

- Install forge from the directory where you want to run your server:
  java -jar forge-1.18.2-40.1.85-installer.jar --installServer
- Edit the 'user_jvm_args.txt' file and set the max memory limit for the server. Recommended value of 8G or more.
- Execute the run.sh or run.bat for your respective environment and accept the EULA.
